# PRODIGY_WD_01

## 🚀 Interactive Navigation Menu - Task 01

### 📝 Description
A responsive landing page with an interactive navigation menu created as part of Prodigy InfoTech Web Development Internship.

### ✨ Features
- ✅ Fixed navigation menu that stays visible while scrolling
- ✅ Dynamic color change on scroll
- ✅ Smooth hover effects on menu items
- ✅ Active section highlighting
- ✅ Smooth scrolling between sections
- ✅ Fully responsive design for all devices
- ✅ Modern gradient design with animations
- ✅ Glass-morphism effects

### 🛠️ Technologies Used
- **HTML5** - Structure
- **CSS3** - Styling with gradients, animations, and effects
- **JavaScript** - Interactivity and smooth scrolling

### 🎨 Design Features
- Animated gradient backgrounds
- Glass-morphism navbar effect
- Smooth transitions and hover animations
- Floating particle effects
- Responsive design for mobile, tablet, and desktop
